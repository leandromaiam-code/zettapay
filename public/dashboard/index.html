How it works
  1. Phantom returns your wallet pubkey.
  2. Server issues a one-time challenge bound to merchant + nonce + 5 min TTL.
  3. You sign the message; server verifies the Ed25519 signature.
  4. Session token (HMAC-signed, 30 min TTL) unlocks the dashboard.
