How it works

Built for three kinds of payers

One protocol. Three integrations. Whether your customer is human, autonomous, or programmatic — ZettaPay just works.

AI Agents

Claude, GPT, Gemini. Pay autonomously via signed transactions in HTTP headers. No keys to custody.

# x402 header with signed Solana tx
POST /pay
x-402-payment: "<signed-tx-blob>"

{
  "merchant": "@coffee-bot",
  "amount_usdc": 2.50
}

Humans

Phantom wallet binding. Stripe-grade UX. Card → USDC via MoonPay onramp built-in.

// Connect Phantom & pay USDC
const { tx } = await zettapay.pay({
  merchant: "@store",
  amount: 42.00,
  source: "phantom"
});

Merchants

Drop-in SDK. Webhooks Stripe-grade with HMAC signing. 3-line integration.

import { ZettaPay } from '@zettapay/sdk';

const zp = new ZettaPay(apiKey);
await zp.merchants.register({
  name: "My Store"
});

For AI Agents

Your agent buys while you sleep

ZettaPay is the first payment processor with native x402 protocol and Model Context Protocol support. AI agents discover, pay, and settle autonomously — no human in the loop.

01
x402 HTTP Payment Protocol
Anthropic's open spec. Sign once, pay anywhere.
02
MCP Tool Definitions
Drop into Claude/GPT instantly. pay, verify, refund.
03
Idempotency & Replay Safety
Agents retry. We dedupe. No double-spend.
claude-agent.ts 
// Define the MCP payment tool
const tools = [{
  name: "pay_with_zettapay",
  description: "Send USDC to a merchant",
  input_schema: {
    type: "object",
    properties: {
      merchant: { type: "string" },
      amount: { type: "number" }
    }
  }
}];

// Claude calls it autonomously
const result = await claude.messages.create({
  tools,
  messages: [{
    role: "user",
    content: "Buy 1 cappuccino from @cafe-bot"
  }]
});

// → ZettaPay receives x402 header
// → Solana tx confirmed in 1.4s
// → Webhook fires to merchant ✓
SDK

Three lines. One payment.

TypeScript SDK with full type safety. npm install and ship.

Install
$ npm install @zettapay/sdk
Initialize
import { ZettaPay } from '@zettapay/sdk';
const zp = new ZettaPay('sk_live_...');
Charge
const payment = await zp.pay({
  merchant: "@my-store",
  amount_usdc: 42.00,
  webhook_url: "https://my-app.com/zp-webhook",
  metadata: { order_id: "ord_xyz" }
});

// payment.signature → Solana tx hash
// payment.status    → "confirmed" within 2s
// Webhook fires to your URL with HMAC-SHA256 signature
