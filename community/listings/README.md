# Solana ecosystem listings

Submission-ready application drafts for the five public Solana ecosystem
surfaces ZettaPay should appear on. Each file is a self-contained payload
written to be copy-pasted into the target form, issue tracker, or PR
description with no further editing required.

| # | Target | File | Submission surface |
| - | ------ | ---- | ------------------ |
| 1 | Anza grants | [`anza-grants.md`](./anza-grants.md) | <https://www.anza.xyz/grants> form |
| 2 | Solana Pay registry | [`solana-pay-registry.md`](./solana-pay-registry.md) | PR to [`anza-xyz/solana-pay`](https://github.com/anza-xyz/solana-pay) |
| 3 | awesome-solana | [`awesome-solana.md`](./awesome-solana.md) | PR to [`avareum/awesome-solana`](https://github.com/avareum/awesome-solana) |
| 4 | RPC providers | [`rpc-providers.md`](./rpc-providers.md) | Email + partnership forms (Helius, Triton, QuickNode, Alchemy, Chainstack) |
| 5 | solana.com / Superteam | [`solana-ecosystem.md`](./solana-ecosystem.md) | <https://solana.com/ecosystem> + Superteam Earn |

## Maintenance rules

- One file per target. Do not consolidate.
- Each file leads with a `Submission checklist` so the operator can
  confirm what to attach (logo, screenshots, audit links) before sending.
- Keep tone factual: feature → on-chain proof → docs link. No marketing
  adjectives.
- When a listing goes live, append the public URL under the file's
  `Status` line and set the date.
- Re-run the awareness sweep every quarter; new RPC providers and
  registries appear faster than this checklist.

## Why these five

- **Anza grants** — funding + signal from the team maintaining the Solana
  validator client. Highest credibility surface.
- **Solana Pay registry** — canonical merchant/wallet directory for the
  Solana Pay URI scheme. ZettaPay implements the scheme natively
  (Z27.3) so qualifies for inclusion.
- **awesome-solana** — long-tail discovery; `awesome-*` lists still drive
  meaningful inbound for OSS infra.
- **RPC providers** — Helius, Triton, QuickNode, Alchemy, Chainstack
  publish "ecosystem partners" pages. Listing there yields free traffic
  AND is a prerequisite for joint customer support.
- **solana.com/ecosystem + Superteam** — official Solana Foundation
  ecosystem directory + Superteam grants/bounties surface. Required for
  any future Solana Foundation co-marketing.
