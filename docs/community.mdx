The ZettaPay community lives on Discord. It's where merchants, AI agent
builders, and SDK users get unblocked, share their integrations, and
hear about releases first.

## Join the server

<Card title="ZettaPay Discord" icon="discord" href="https://discord.gg/zettapay">
  Free, public, English + Portuguese friendly.
</Card>

## What's where

| Channel | When to use it |
| --- | --- |
| `#welcome` | Read first. Use `/role` to pick what describes you (Merchant, Agent Builder, Developer). |
| `#announcements` | Releases, mainnet milestones, audit results, security advisories. |
| `#help` | Stuck on integration — RPC, USDC mint, x402, webhooks, signatures. |
| `#showcase` | Built something with `@zettapay/sdk` or x402? Share a demo or write-up. |
| `#api` | REST + SDK questions: `POST /pay`, idempotency, rate limits. |
| `#x402` | The agent payment protocol — headers, signed tx blobs, facilitators. |
| `#mcp` | Wiring ZettaPay as an MCP tool for Claude, GPT, Gemini. |
| `#webhooks` | Signature verification, retries, idempotency keys. |
| `#on-chain` | Solana program, IDL, Anchor, devnet/mainnet. |
| `#feedback` | Product feedback, feature requests, DX papercuts. |

## Code of conduct

Be respectful. Stay on topic per channel. No financial advice, no token
shilling, no airdrop bait. Never share private keys, seed phrases, or API
keys — moderators will never DM first asking for credentials.

The full rules live in `#rules` on the server.
