## `ZettaPayClient`

```ts
new ZettaPayClient(opts: ZettaPayClientOptions)

interface ZettaPayClientOptions {
  baseURL: string;
  timeoutMs?: number;
  headers?: Record<string, string>;
  axiosInstance?: AxiosInstance;
}
```

### Methods

| Method | Signature | Returns |
| --- | --- | --- |
| `pay` | `(input: PayInput \| string \| Uint8Array)` | `PayResponse` |
| `registerMerchant` | `(input: RegisterMerchantInput)` | `Merchant` |
| `getMerchant` | `(id: number)` | `Merchant` |
| `listMerchants` | `(opts?: ListMerchantsOptions)` | `ListMerchantsResponse` |
| `updateMerchant` | `(id: number, patch: Partial<Merchant>)` | `Merchant` |
| `deleteMerchant` | `(id: number)` | `void` |
| `getPayment` | `(id: string)` | `PaymentRecord` |
| `listPayments` | `(opts?: ListPaymentsOptions)` | `ListPaymentsResponse` |
| `health` | `()` | `HealthStatus` |

## On-chain helpers

| Helper | Signature |
| --- | --- |
| `deriveMerchantBindingPda` | `(handle: string, owner: PublicKey) => { pda: PublicKey; bump: number }` |
| `derivePaymentPda` | `(merchantBinding: PublicKey, paymentId: string) => { pda: PublicKey; bump: number }` |
| `buildRegisterMerchantInstruction` | `(params: BuildRegisterMerchantParams) => TransactionInstruction` |
| `buildRecordPaymentInstruction` | `(params: BuildRecordPaymentParams) => TransactionInstruction` |
| `registerMerchantOnChain` | `(params: RegisterOnChainParams) => Promise<{ signature: string; pda: PublicKey }>` |
| `recordPayment` | `(params: RecordPaymentParams) => Promise<{ signature: string; pda: PublicKey }>` |
