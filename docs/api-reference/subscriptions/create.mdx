---
title: "Create subscription"
description: "Schedule a recurring USDC charge against a customer wallet."
api: "POST https://api.zettapay.io/subscriptions"
---

Subscriptions charge a payer wallet on a recurring cadence. The first
charge fires at `nextChargeAt` (defaulting to one interval from now) and
recurs until the subscription is canceled or hits a permanent failure.

## Headers

| Header | Required |
| --- | --- |
| `x-zettapay-api-key` | Yes |
| `Idempotency-Key` | Recommended |
| `Content-Type` | Yes |

## Body

| Field | Type | Required | Notes |
| --- | --- | --- | --- |
| `customerWallet` | string | Yes | Solana base58 pubkey to debit. |
| `amount` | string | Yes | Decimal USDC per charge, e.g. `"19.99"`. |
| `interval` | string | Yes | `"daily"`, `"weekly"`, or `"monthly"`. |
| `currency` | string | No | Defaults to `"USDC"`. |
| `nextChargeAt` | string | No | ISO-8601 UTC. Defaults to `now + interval`. |
| `metadata` | object | No | Returned on every webhook fired by this subscription. |

## Example request

```bash
curl -X POST https://api.zettapay.io/subscriptions \
  -H "x-zettapay-api-key: zp_live_..." \
  -H "Idempotency-Key: sub_setup_01J9VV..." \
  -H "Content-Type: application/json" \
  -d '{
    "customerWallet": "CustomerPhantomPubkey...",
    "amount": "19.99",
    "interval": "monthly",
    "metadata": { "plan": "pro" }
  }'
```

## Response — `201 Created`

```json
{
  "subscription": {
    "id": "sub_01J9VV...",
    "merchantId": 42,
    "customerWallet": "CustomerPhantomPubkey...",
    "amount": "19.99",
    "currency": "USDC",
    "interval": "monthly",
    "status": "active",
    "nextChargeAt": "2026-06-10T19:14:02.000Z",
    "metadata": { "plan": "pro" },
    "createdAt": "2026-05-10T19:14:02.111Z"
  }
}
```
