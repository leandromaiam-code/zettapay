# 06 · Vue 3 + `<zetta-checkout>` web component

Same idea as the React example, in Vue. The web component does the heavy lifting — your Vue layer just composes it.

## Run

```bash
cp .env.example .env
npm install
npm run dev
```

## Files

- `src/ZettaCheckout.vue` — single-file component wrapping the embed.
- `src/App.vue` — demo page.
