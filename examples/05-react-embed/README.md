# 05 · React + `<zetta-checkout>` web component

Wraps the ZettaPay embed (a framework-agnostic web component) in a React component so it composes with your existing app.

## Run

```bash
cp .env.example .env
npm install
npm run dev
```

## Files

- `src/ZettaCheckout.tsx` — typed React wrapper.
- `src/App.tsx` — demo page.
- `index.html` — Vite entry.
